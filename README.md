# Encrypter Tool

A simple custom text encryption–decryption tool built from scratch.  
It allows you to encrypt any text with a password and decrypt it back only with the correct password.

## Features

- Encrypt text with a password
- Decrypt text with the same password
- Standalone `.exe` version for Windows
- Pure offline computation, no internet required

## Getting Started

1. Download the `Encrypter.exe` file from the `dist` folder.  
2. Run it locally on your Windows machine.  
3. Enter your password and text to encrypt/decrypt.

## Usage

- Enter any text or encrypted numbers.  
- Provide a password.  
- Choose Encrypt or Decrypt.  
- Copy the output.
